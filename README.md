# 💰 Деньги — София и Никита

Простое мобильное веб-приложение для двоих: выбираешь, кто ты (София или
Никита), и можешь **переписываться в чате** и **переводить деньги** друг
другу. Балансы и вся история переводов сохраняются.

- Начальные балансы: **Никита — 298 D**, **София — 250 D**
- Интерфейс на русском, оптимизирован под телефон
- Без внешних зависимостей — только Node.js

## Запуск

```bash
node server.js
```

Затем открой в браузере `http://localhost:3000`.

Порт можно изменить переменной окружения `PORT`:

```bash
PORT=8080 node server.js
```

## Как пользоваться

1. При входе выбери, кто ты — **София** или **Никита**.
   Выбор запоминается в браузере (кнопка «Сменить» наверху меняет пользователя).
2. **💬 Чат** — переписка между вами в реальном времени.
3. **💸 Перевод** — отправить сумму партнёру (с комментарием).
4. **📜 История** — все переводы: входящие (+) и исходящие (−).

Данные обоих устройств синхронизируются через сервер каждые 3 секунды.

## Хранение данных

Всё состояние (балансы, сообщения, переводы) хранится в файле `data.json`
рядом с сервером. При первом запуске файл создаётся автоматически с
начальными балансами. Чтобы сбросить всё к началу — удали `data.json`.

> Файл `data.json` не попадает в git (см. `.gitignore`), чтобы реальная
> история не уезжала в репозиторий.

```

public/index.html  — разметка
```

## API

| Метод | Путь            | Тело                          | Описание              |
|-------|-----------------|-------------------------------|-----------------------|
| GET   | `/api/state`    | —                             | Балансы, чат, история |
| POST  | `/api/message`  | `{ from, text }`              | Отправить сообщение   |
| POST  | `/api/transfer` | `{ from, amount, note }`      | Перевести деньги      |
