# 💰 Деньги — София и Никита

Простое мобильное веб-приложение для двоих: выбираешь, кто ты (София или
Никита), и можешь **переписываться в чате** и **переводить деньги** друг
другу. Балансы и вся история переводов сохраняются.

- Начальные балансы: **Никита — 298 D**, **София — 250 D**
- Интерфейс на русском, оптимизирован под телефон
- Без внешних зависимостей — только Node.js

## Запуск

```bash
node server.js
```

Затем открой в браузере `http://localhost:3000`.

Порт можно изменить переменной окружения `PORT`:

```bash
PORT=8080 node server.js
```

## Как пользоваться

1. При входе выбери, кто ты — **София** или **Никита**.
   Выбор запоминается в браузере (кнопка «Сменить» наверху меняет пользователя).
2. **💬 Чат** — переписка между вами в реальном времени.
3. **💸 Перевод** — отправить сумму партнёру (с комментарием).
4. **📜 История** — все переводы: входящие (+) и исходящие (−).

Данные обоих устройств синхронизируются через сервер каждые 3 секунды.

## Хранение данных

Состояние (балансы, сообщения, переводы) хранится двумя способами:

- **Локально** — в файле `data.json` рядом с сервером (создаётся
  автоматически при первом запуске). Чтобы сбросить всё к началу —
  удали `data.json`.
- **В облаке (Upstash Redis)** — если заданы переменные окружения
  `UPSTASH_REDIS_REST_URL` и `UPSTASH_REDIS_REST_TOKEN`. Тогда данные
  хранятся постоянно и переживают перезапуски сервера (нужно для
  бесплатного Render — см. ниже).

> Файл `data.json` не попадает в git (см. `.gitignore`).

## Публикация на Render (бесплатно)

Чтобы пользоваться с телефонов из любого места:

### 1. Постоянное хранилище — Upstash Redis (бесплатно, без карты)

1. Зарегистрируйся на <https://upstash.com> (можно через GitHub).
2. Создай базу **Redis** (Create Database), регион — ближе к тебе.
3. На странице базы найди блок **REST API** и скопируй два значения:
   - `UPSTASH_REDIS_REST_URL`
   - `UPSTASH_REDIS_REST_TOKEN`

### 2. Хостинг — Render

1. Зайди на <https://render.com> через GitHub.
2. **New → Web Service** и выбери этот репозиторий.
3. Render сам подхватит `render.yaml`. Настройки по умолчанию:
   - Runtime: **Node**, план: **Free**
   - Start command: `node server.js`
4. В разделе **Environment** добавь две переменные из Upstash:
   `UPSTASH_REDIS_REST_URL` и `UPSTASH_REDIS_REST_TOKEN`.
5. Нажми **Deploy**. Через пару минут получишь адрес вида
   `https://dengi-xxxx.onrender.com` — открывай его на телефоне.

> Без переменных Upstash на бесплатном Render данные сбрасываются, когда
> сервис «засыпает». С Upstash балансы и история сохраняются навсегда.

> Бесплатный Render усыпляет сервис после простоя — первая загрузка
> после сна занимает ~30 секунд, дальше работает быстро.

## API

| Метод | Путь            | Тело                          | Описание              |
|-------|-----------------|-------------------------------|-----------------------|
| GET   | `/api/state`    | —                             | Балансы, чат, история |
| POST  | `/api/message`  | `{ from, text }`              | Отправить сообщение   |
| POST  | `/api/transfer` | `{ from, amount, note }`      | Перевести деньги      |
